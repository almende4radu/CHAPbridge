Welcome to CHAPbridge!

This is the webpage of CHAPbridge project, which illustrates CHAP use for BRIDGE domain.

To obtain the documentation, run the following commands.

$ cd github.com/CHAPbridge
$ git fetch origin
$ git checkout gh-pages

You now have the new branch.

Manual documentation using jekyll, markdown

This project was documented manually by pushing a branch named gh-pages to the repository. GitHub Pages support Jekyll, a simple, blog aware static site generator. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
